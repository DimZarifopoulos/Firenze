# MyCityGuide - Android City Tour Application

MyCityGuide is an Android application designed to serve as a digital tour guide for city visitors. It highlights specific points of interest (POIs), provides interactive details, and engages users with educational content.

## Key Features

### 📍 Points of Interest (POIs)
* The app features at least 5 POIs, each including:
    * Title and Category (e.g., museum, monument, square).
    * Detailed description and a representative image.
    * Geographical coordinates and a "Learn more" URL.

### 🗺️ Map Integration
* An interactive map displays markers for all POIs.
* Clicking on a marker navigates the user directly to the details page of the specific monument.

### 🔊 Accessibility & Details
* The details page includes an "Listen" button that utilizes Text-to-Speech (TTS) to narrate the POI description.
* Integrated web view to load external resources (e.g., Wikipedia links) directly within the app.

### 📊 Database & User Statistics
* A local database stores user information (name, age) and tracks usage counters (how many times each POI has been opened or listened to).
* Dedicated "Statistics" screen to view user data and engagement metrics.

### 🧠 User Engagement
* After viewing or listening to a POI, users can take a multiple-choice quiz to test their knowledge, with a final score display.

## Technical Implementation
* **Development Tools:** Developed using either MIT App Inventor or Android Studio.
* **Code Organization:** Built with a focus on reusable procedures and clean code practices to minimize repetition.
